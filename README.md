# Online E-Commerce Store (DSA Project)

## Overview
This project is a console-based online e-commerce store system developed as part of a Data Structures and Algorithms (DSA) course. It demonstrates how fundamental DSA concepts can be applied to simulate core functionalities of an e-commerce platform such as product management, order handling, and delivery optimization.

The primary focus is on efficient data organization and algorithmic problem-solving rather than building a full-scale commercial application.

## Features
- View available products  
- Place and manage orders  
- Store and retrieve customer details  
- Efficient product searching  
- Delivery route optimization  

---

## Data Structures & Algorithms Used

### Linked List
Used to manage:
- Customer records  
- Order history  

### AVL Tree / Binary Search Tree (BST)
Used for:
- Storing and searching products efficiently  
- Maintaining sorted inventory  

### Graph
Used to:
- Represent delivery routes between different locations  

### Dijkstra’s Algorithm
- Finds the shortest path for delivery between two locations  

### Prim’s Algorithm
- Determines minimum delivery cost using Minimum Spanning Tree
